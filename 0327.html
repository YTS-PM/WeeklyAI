📉 本週趨勢摘要 (Executive Summary)

📊 關鍵細節
  • 算力成本過高：影片生成所需的訓練與推論成本極度昂貴，嚴重排擠了其他核心文本與邏輯推論模型的算力配額。

Anthropic Claude能直接操控電腦執行任務了！網驚：秒殺了OpenClaw

#ComputerUse#AgenticAI#自動化革命
⚡ 重點速報
Anthropic 推出了一項顛覆性的「Computer Use」功能，賦予 Claude 大模型直接「觀看」螢幕、移動游標、點擊按鈕與輸入文字的能力。這意味著 Claude 不再需要依賴特定的 API 串接，它能像一個真實的人類員工一樣，跨越各種未開放 API 的古老軟體（如傳統銀行的核心系統或內部表單）完成端到端的任務執行。
📊 關鍵細節
  • 視覺化操作：Claude 可即時截取螢幕畫面，理解 GUI（圖形使用者介面）元素，並計算游標座標以進行精準點擊。
  • 跨應用作業：打破軟體壁壘，Claude 可以自主打開網頁瀏覽器搜尋資料、複製內容，接著打開 Excel 整理，最後再發送 Email。
  • 顛覆舊有框架：市場驚呼此技術的泛用性直接威脅了依賴特定開源框架（如 OpenClaw）或傳統 RPA（機器人流程自動化）的舊有解決方案。
💡 策略解讀

這是 RPA（機器人流程自動化）的終結者，也是金融業後台營運的核彈。過去我們為了自動化，需要 IT 部門耗費數月去串接不同舊系統的 API，若無 API 則束手無策。Claude 的 Computer Use 帶來了「暴力解法」：只要人能透過介面操作的，AI 就能操作。數位金融長（CDO）應立即盤點內部大量「跨系統的人工複製貼上」流程（如徵信資料收集、洗錢防制審查），將其列為第一波 AI 代理人接管的重點實驗區。

Anthropic推出新功能「Dispatch」：用手機就能對Claude下指令

#邊緣與雲端協作#生產力工具#Anthropic
⚡ 重點速報
Anthropic 發布了名為「Dispatch」的新功能，徹底打通了行動裝置與桌面工作站的壁壘。使用者在外出時，可透過手機對 Claude 下達複雜的自然語言指令，Claude 隨即會在使用者的桌面電腦端啟動，自動開啟特定檔案、編寫程式碼或製作報表。這強調了跨裝置無縫協作與最高等級的隱私保護。
📊 關鍵細節
  • 遠端喚醒：手機作為「意圖發射器」，桌面端作為「算力與執行終端」，突破了手機螢幕與算力的限制。
  • 隱私保護架構：Dispatch 強調在建立遠端連結時，數據傳輸符合企業級資安標準，不會擅自讀取無關的本地敏感檔案。
  • 企業場景落地：非常適合需要高行動力的業務主管，在見客戶前一秒透過手機下令，回到辦公室時所需的市場分析報告已經在電腦前完成。
💡 策略解讀

Dispatch 展示了未來的金融從業人員工作型態：「行動指揮官」。對於銀行 RM（關係經理）或理專而言，這是一項極具破壞力的工具。我們不應再把 AI 侷限於「網頁版的一個聊天室」，而是將其視為一位隨時候命的「虛擬幕僚」。數位團隊應思考如何將銀行內部的數據平台，打造成能接收這類「跨裝置 Agentic 指令」的架構，讓前線戰力實現非線性增長。

金融科技巨頭 Solaris 裁員 20%，轉型「AI原生銀行」

#組織重整#AI原生#BaaS
⚡ 重點速報
歐洲銀行即服務（BaaS）先驅 Solaris 宣布了一項激進的結構重整計畫：裁員 20% 以削減營運成本，並將資源全面投入轉型為「AI 原生銀行（AI-Native Banking）」。Solaris 的管理層表示，傳統的人力密集營運模式已無法支撐未來金融的敏捷需求，未來的核心架構將由 AI 代理人直接負責合規審查、客戶服務與動態風險定價。
📊 關鍵細節
  • 裁員規模：大刀闊斧削減 20% 人力，主要集中於傳統客服、基礎法遵審核與重複性維運人員。

Solaris 的「斷臂求生」是給全體金融機構的一記警鐘。他們證明了 AI 不僅是「增加收入」的工具，更是「重塑成本結構」的剃刀。當同業能透過「AI 原生架構」將維運成本降低 30% 以上時，他們就能提供更低的貸款利率與更高的存款利息。台灣金融業的主事者應開始思考：我們現在設計的新系統，是為「人類員工」設計的，還是為「未來的 AI 代理人」設計的？

傳蘋果獲Google Gemini底層完整授權 將透過「模型蒸餾」打造Siri專屬在地化小模型

#模型蒸餾#邊緣AI#生態系融合
⚡ 重點速報
市場盛傳 Apple 已成功取得 Google Gemini 的核心底層授權。不同於過去僅以 API 形式呼叫外部模型，蘋果計畫運用「模型蒸餾（Model Distillation）」技術，將龐大的 Gemini 智慧壓縮至適合在 iPhone 端點離線運行的專屬小型語言模型（SLM），徹底重塑 Siri 的對話能力，同時確保無懈可擊的在地化隱私。
📊 關鍵細節
  • 模型蒸餾技術：利用擁有強大能力的「教師模型（Gemini）」來指導「學生模型（Siri端點模型）」，大幅保留推理能力同時將體積縮減 90%。
  • 隱私優先：藉由離線 SLM 運行，使用者的個人財務數據、行事曆與對話內容無需上傳雲端，完美契合蘋果的隱私護城河。
  • 世紀聯手：此舉標誌著科技兩大巨頭在 AI 時代的深度妥協與互利，Google 換取了數十億終端入口，蘋果換取了最先進的基礎模型智商。
💡 策略解讀

「終端 AI（Edge AI）」才是金融業最應該關注的最後一哩路。當客戶的 iPhone 本身就具備強大的本地 AI 推論能力時，未來的「AI 網銀」可能不再需要將敏捷的財務提問發送到銀行的雲端伺服器。數位金融部門應積極研究 Apple Intelligence 的開發者框架（App Intents），準備讓自家銀行的功能可以被更聰明的本地 Siri 順暢調用，搶佔第一波端點入口紅利。

#AI政策#地緣政治#國家戰略
⚡ 重點速報
美國川普政府正式發布最新的國家 AI 政策框架。有別於過往高度強調監管與限制防弊，新框架展現了強烈的「進攻性」，定調 AI 是確保美國全球領導地位的絕對核心。政策主軸涵蓋：大幅放寬國內算力基礎建設的環評限制、對 AI 人才提供戰略性簽證，並以國家資源支援基礎模型研發，確保美國在技術競爭中絕對領先。
📊 關鍵細節
  • 基建解綁：承諾為 AI 資料中心建設提供「快速通關」，降低電力與環評的審查門檻，以解決算力瓶頸。

💡 策略解讀

美國政府的新政策預示著一場「算力去管制化」的狂飆期即將到來。對金融機構而言，這意味著美國雲端供應商（AWS, Azure, GCP）的底層發展阻力降低，算力供給可能比預期更快放量。但同時，地緣政治的切割會更為鋒利，台灣金融業在選擇開源模型或底層算力架構時，必須確保 100% 依附在美國認可的安全供應鏈體系內，避免意外的合規制裁風險。

  • 繁中語料優勢：內建經由高品質台灣本土語料訓練的專屬大模型，大幅降低金融機構使用歐美模型時的文化與法規幻覺。
💡 策略解讀

這對台灣金融業的 IT 部門是一場及時雨。過去金融業在推動生成式 AI 落地時，最大的阻礙不是技術，而是「資料能不能上公有雲」的法規大哉問。AISO 聯盟提供的在地化算力與私有化部署方案，恰好提供了合規的解題路徑。數位與 IT 團隊應主動接洽此類本土解決方案，將高敏感度的業務（如理財健檢、授信模型審查）以主權 AI 架構進行概念驗證（PoC）。
